التكملة
«أيهم» هو مشروع تمت دراستهي-رقمي طويل المدى يهدف إلى بناء عقل رقمي حي ، قائم على التوثيق التاريخي، وتحليل التطبيقاتي، والذاكرة الآشپية، وقابل للتراكم، غير قابل للمحو، وغير بشكل منفصل عن صاحبه.

المشروع لا يمثل تطبيقًا تقنيًا فقط، نظام تفكير حي يتم تنفيذه عبر التجربة، التوثيق، التحليل، والترميز الرمزي.
